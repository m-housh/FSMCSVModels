# FSMCSVModels

Utility package that represents CSV data models for Fieldstone Management Software.

## Usage
---------------

*Package.swift*
```swift

let package = Package(
    ...
    dependencies: [
        .package(url: "https://github.com/m-housh/FSMCSVModels.git", from: "0.1.0")
    ],
    ...
)

```
